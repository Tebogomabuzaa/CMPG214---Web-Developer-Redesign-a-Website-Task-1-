## Document Summary

The final specification covers:

- **Current site evaluation** — five identified usability weaknesses including readability, fixed-width layout failure, button affordance, mobile cookie badge obstruction, and low-visibility action links
- **Redesign objectives** — four objectives mapped directly to the weaknesses above
- **Proposed changes** — fluid layout, persistent Donate button with overlay, alternating high-contrast sections, and a consistent button colour system
- **Design rationale** — user experience justification for each decision
- **Implementation considerations** — accessibility, responsive typography, semantic HTML, and component-based architecture

Word count: 1 000 words (body text).
